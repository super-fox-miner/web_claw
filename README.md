## 📌 项目简介

这是一个基于FastAPI和MCP（Model Context Protocol）协议构建的服务器应用，为AI模型提供标准化的文件系统操作和Office文档（Word、Excel）文档处理能以及PDF阅读能力。

本项目90%代码由AI生成   使用trae编辑器编写

本项目想力争解决现阶段AI使用两大问题：
1. 使用api-key调用的智能体成本问题，以OpenClaw为代表，token花费非一般人能接受
2. 网页版没有本地文件系统操作能力，且不具备记忆功能，每次新会话就要重新要求ai规则，无法做到“越用越智能”的效果

## ✨ 核心特性

### 养龙虾模式
在此模式下可以为网页版的AI定制规则，比如文档编写规范或者要求每次对话都要符合一定的格式。

### 龙虾出征模式
在此模式下，AI会根据你之前指定的规则内容进行任务的处理，做到“越用越智能”的效果。

### 本地化文件系统操作
目前项目包含了一个自制的文件操作系统以及三个第三方的开源MCP，主要针对word，excel相关操作和pdf阅读功能。

## 📦 快速开始

### 环境要求
- Python 3.11+

### 安装步骤
1. 在浏览器插件管理中安装篡改猴，然后将项目文件夹中的JS文件加载到浏览器中。（目前只支持豆包和deepseek）
2. 双击运行`start.bat`即可启动服务。（第一次运行会安装配置）

## ⚙️ 配置文件详解

### .env - 环境配置
```
# 服务器配置
SERVER_HOST=0.0.0.0
SERVER_PORT=8080

# 文件系统安全目录
在.env中配置，例如`BASE_DIR=d:/LocalMind`

# 向量模型配置
EMBEDDING_MODEL_PATH=./models/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2

```
