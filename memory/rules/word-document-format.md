# Word文档格式规则

## 标题格式
居中，2号方正小标宋，不加粗。可分一行或多行居中排布；回行时，要做到词意完整，排列对称，间距恰当。标题一般要有行文单位。

## 正文格式

### 一级标题
（一、……），3号黑体，不加粗，句末不加标点符号，（在序号后是、）然后回行。

### 二级标题
（（一）……），3号楷体，不加粗，句末不加标点符号，（在序号后无标点）然后回行。

### 三级标题
（1.……），3号仿宋，加粗，句末不加标点符号，(序号后面是. 不是，也不是、)然后回行。

### 其余标题
（（1）……，①……，A、……）与正文要求相同，不回行。

## 正文段落
正文，3号仿宋，一般每面排22行，每行排28个字。

### 页面设置
- 上边距：3.7cm
- 下边距：3.4cm  
- 左边距：2.8cm
- 右边距：2.3cm
- 行间距：固定值28磅

### 段落规则
每自然段左空2字，回行顶格；数字、年份不能回行。具体排版时，根据文字、段落、格式需要可适当调整。
